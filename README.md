# 🚀 Customer Lifetime Value Prediction (ML Project)

## 📌 Overview
This project predicts Customer Lifetime Value (CLTV) using machine learning.

## 🎯 Problem Statement
Identify high-value customers using their data and policy information.

## 🧠 Approach
- Data preprocessing  
- Feature engineering  
- XGBoost model  
- Validation to avoid overfitting  

## 📊 Results
- R² Score: **0.155+**
- Rank: **Top 100 (#64)**

## 🛠️ Tech Stack
- Python  
- Pandas  
- Scikit-learn  
- XGBoost
